Speaker Recognition Toolkit – Requirement Specifications


Modules

  1. Data processing

    1. Listing out wav files

    2. Checking sampling frequency

    3. Normalization: Mean-Max

    4. VAD – segmentation to subutterance: Chop long utterances: Development and enrollment utterances are chopped into 50s segments. Test utterances are chopped into 10-60s segments

    5. Feature Computation - MFCC

    6. Feature vector selection w.r.t VAD

  2. Feature extraction,

  3. Background model training (GMM-UBM, TV, TDNN-X-vector, custom model),

  4. Enrolment (i-vector, x-vector, embedding vector extractor ),

  5. post-processing (LDA, WCCN, normalization),

  6. Front-end classifier (PLDA, cosine distance),

  7. Inference and API

  8. Visualization (ROC, DET curves)
